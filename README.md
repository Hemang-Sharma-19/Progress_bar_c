# Progress Bar in C

This project is a simple terminal-based progress bar simulation written in C. It demonstrates multiple tasks progressing simultaneously until each task reaches 100%.

## Description

The program generates random progress for multiple tasks and displays animated progress bars in the terminal. The screen is refreshed continuously to create a smooth progress animation.

## Features

- Multiple task progress simulation
- Animated progress bars
- Random progress increments
- Automatic completion detection
- Console-based interface

## Technologies Used

- C
- Standard C Library
